# 빌드 방법

## 설치

```bash
$ git clone git주소
$ yarn or npm install
$ yarn start or npm run start
```

## 빌드
- ECMAScript 6 지원 브라우저 사용
- 권장: Google Chrome 버젼 77 이상

# 기술 스택

### Server(back-end)

-  nodejs, php, java 등 서버 언어 버전 
- express, laravel, sptring boot 등 사용한 프레임워크 
- DB 등 사용한 다른 프로그램 
 
### Front-end

 -  react.js, vue.js 등 사용한 front-end 프레임워크 
 -  UI framework
 - 기타 사용한 라이브러리
